# Moose Farm Bot — финальный патч WizeBot-фич

Добавлено без обратного sync сайт -> WizeBot:

- Overlay/визуал открытия кейса на сайте.
- Расширенные редкие админ-команды:
  - удалить турель игрока;
  - создать backup перед опасными действиями;
  - восстановить последний backup;
  - установить билеты рулетки 0-150;
  - сформировать 1:1 чеклист для живой проверки.
- Чеклист 1:1 по блокам:
  - стоимость апов;
  - стоимость зданий;
  - доход зданий;
  - рейды;
  - турель;
  - рынок;
  - кейсы;
  - GAMUS;
  - оффсбор;
  - админ-действия.
- Новые события в админ-журнале:
  - admin_delete_turret;
  - admin_restore_backup;
  - admin_set_roulette_tickets;
  - admin_1to1_check.

Важно:
- Полный перенос старых WizeBot custom variables в сайт возможен только из данных, которые уже попали в SQLite через sync/import. Прямой массовый перенос всех WizeBot-переменных без отдельного WizeBot API/export не включён.
- Обратный sync сайт -> WizeBot не трогался.
- Авто-sync при обновлении страницы не включался.
