# Moose Farm UX/Admin/Safety Fix

Что сделано:

- Убран длинный блок админки из основного скролла. Теперь админка открывается отдельной кнопкой `👑 Открыть админ-панель`.
- Кнопка админки видна только `Nico_Moose`; backend по-прежнему защищён `requireAdmin`.
- Админка разбита по вкладкам: Баланс, Уровни, Опасное, Расширенное, Журнал.
- Добавлен поиск игрока через выпадающий список: вводишь начало ника, список фильтруется по `login/display_name`.
- Добавлен endpoint `GET /api/admin/players?prefix=N`.
- Добавлена защита от двойных кликов:
  - frontend блокирует одинаковые повторные POST-действия;
  - backend блокирует одновременные POST-запросы к `/api/farm/*` для одного игрока;
  - admin backend блокирует одновременные POST-запросы для одного админ-действия/цели.
- Бонусный баланс через админку больше не уходит ниже 0.
- Стили UI приведены к более компактным категориям, чтобы меньше листать вниз.

Не трогалось:

- Обратный sync сайт → WizeBot.
- Авто-sync при обновлении страницы.
- Формулы экономики/рейдов/кейсов.
