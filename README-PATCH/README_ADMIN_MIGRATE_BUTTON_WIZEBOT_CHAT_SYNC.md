# PATCH: админ-кнопка «Мигрферма» для legacy !купитьферму

## Что сделано
- В карточке игрока в админ-панели рядом с кнопкой `↻ Обновить` добавлена красная кнопка `🟥 Мигрферма`.
- Кнопка вызывает новый backend-route `POST /api/admin/trigger-legacy-migration`.
- Route отправляет в Twitch chat команду WizeBot-миграции через бота (`!сайтмигрферма <login>` по умолчанию).
- После отправки route ждёт, пока сайт реально получит свежие данные из WizeBot, и только потом отвечает успехом.
- В статусе админ-панели показывается сообщение `WizeBot данные синхронизированы: <login>`.

## Важно
- Команда, которую бот пишет в чат, берётся из `WIZEBOT_LEGACY_MIGRATION_COMMAND`.
- Если переменная не задана, используется `!сайтмигрферма`.
- Поэтому, если в чате у тебя команда называется иначе (например `!мигрферма`), просто задай:
  - `WIZEBOT_LEGACY_MIGRATION_COMMAND=!мигрферма`

## Почему это нужно
Это удобно для игроков, у которых ферма куплена через старую команду `!купитьферму` и потом прокачивалась в legacy WizeBot. Теперь админ может открыть игрока на сайте и одной кнопкой принудительно подтянуть актуальные данные в сайт/farm_v2.
