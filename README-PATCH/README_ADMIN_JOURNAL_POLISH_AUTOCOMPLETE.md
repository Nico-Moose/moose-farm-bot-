# Admin journal polish + autocomplete

Точечный патч только для журнала админ-панели.

Изменено:
- добавлен красивый блок фильтров журнала;
- поиск игрока в журнале теперь подхватывает игроков фермы по нику через существующий `/api/admin/players`;
- при выборе игрока журнал сразу загружается по этому игроку;
- журнал ограничен последними 7 днями через `days=7`;
- карточки событий в админ-журнале сделаны читабельнее;
- остальные вкладки админки не менялись.

Файлы для замены:
- `public/farm.html`
- `public/app.js`
- `public/style.css`
- `routes/adminRoutes.js`
