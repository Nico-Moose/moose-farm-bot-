# Большой UI/UX patch

Дата: 2026-05-03

Что добавлено/доработано:
- Единый компонент модалок `unifiedModal` и `renderUnifiedReward`.
- Единый стиль отчётов для кейса, GAMUS, оффсбора и будущих рейд-репортов.
- Улучшенная история кейсов в формате таблицы: дата, тип, выигрыш, множитель, цена.
- Доработанный рынок: компактный калькулятор, история сделок, понятные ограничения покупки/продажи.
- Доработанная вкладка топов/инфо: отдельные карточки по голде, ферме, бонусным, запчастям, активности и рейдам.
- Отдельные карточки зданий в топ/инфо с описанием, что даёт каждое здание.
- Мягкий UI для backup/restore в админке: список backup’ов, preview JSON, выбор блока восстановления.
- Дополнительный визуальный полиш и мобильная адаптация.

Важно:
- Patch добавлен безопасными override-функциями в конец `public/app.js`.
- Если в проекте есть старые функции с теми же именами, последние объявления в файле имеют приоритет.
- Backup/restore UI использует существующие endpoints `/api/admin/backups` и `/api/admin/restore-backup-index`.
- Для полноценной production-админки можно позже вынести backup UI в отдельный модуль.
