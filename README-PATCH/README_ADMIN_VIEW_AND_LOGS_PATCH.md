# Admin View + Logs Patch

Дата: 2026-05-03

Что сделано:
- Добавлена маленькая красивая кнопка “вверх” справа сверху сайта.
- В поле админского ника добавлен крестик для быстрой очистки и выбора нового игрока.
- В админке добавлена кнопка “Смотреть профиль игрока”.
- Админ может открыть красивый preview профиля игрока: балансы, уровень, лицензия, защита, рейд-сила, турель, постройки, кейсы, последние рейды.
- Расширенное меню админки группируется по категориям:
  - Стрим и оффсбор
  - Перенос и синхронизация
  - Экономика и рынок
  - Тесты и чеклисты
  - Остальное
- Карточка выбранного игрока в админке стала компактнее и удобнее.
- Backend-логи для raid/case/GAMUS/offcollect дополнены человекочитаемыми полями `eventTitle` и ключевыми числами.
- Это помогает журналу меньше показывать сырой JSON и больше показывать понятные действия.

Что ещё остаётся:
- Полный “войти как игрок” без смены сессии не сделан: сейчас это админский preview в модалке.
- Для идеального raid breakdown нужно, чтобы `raidService` сам отдавал поля `money_from_main`, `money_from_farm`, `debt_after`, `turret_chance`, `jammer_level`.
- Журнал станет полностью идеальным, когда все сервисы будут логировать события в едином формате, а не только основные действия.
