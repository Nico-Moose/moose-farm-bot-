# SAFE PATCH — Premium loot UI rules update

Архив подготовлен под текущий проект `moose-farm-bot--main (30)` и совместим с уже поставленным premium loot / multi-take UI.

## Что меняется

1. **Ссылка поддержки** заменена на:
   - `https://donatex.gg/donate/nico_moose`

2. **Верхние карточки профиля** ужаты и сделаны аккуратнее:
   - уменьшены размеры текста и внутренние отступы,
   - добавлена защита от некрасивого переполнения,
   - кнопки/карточки должны влезать лучше.

3. **Инвентарь предметов теперь хранит и показывает отдельные иконки без нежелательного склеивания наград между открытиями**:
   - новые открытия больше не мерджатся в одну запись инвентаря,
   - одинаковые предметы сортируются рядом,
   - но остаются отдельными selectable-иконками.

4. **Правила раскладки предметов по иконкам**:
   - `Bolt`, `L96`, `M4`, `M249` → по **1 иконке на каждый предмет**;
   - `Jagger`, `Silencer` → по **1 иконке**;
   - `Smoke` → всегда по **6 шт на 1 иконку**;
     - пример: `Smoke x18` => **3 иконки по x6**;
   - `Incendiary 5.56` → **стак как выпал**;
     - пример: `x16`, `x32`, `x64` остаются отдельными стаками;
   - `Drone` убран;
   - вместо него используется **Max Health Tea**;
     - там где раньше было `2 drone` → теперь `Max Health Tea x4`,
     - там где раньше было `5 drone` → теперь `Max Health Tea x8`.

5. **Групповое забирание**:
   - можно выбрать один или несколько предметов,
   - при выдаче результат агрегируется красиво,
   - пример: если выбрать 3 `M249`, в выдаче/оверлее уйдёт `M249 x3`.

6. **OBS overlay**:
   - для чая добавлена локальная картинка `public/img/maxhealthtea.pure.png`;
   - overlay продолжает работать через уже сделанный event-bus.

## Как установить

1. Заменить файлы из архива в проекте.
2. Перезапустить сайт / node-процесс.
3. Сделать `Ctrl + F5` в браузере.
4. Проверить:
   - ссылка поддержки ведёт на `https://donatex.gg/donate/nico_moose`;
   - предметы в инвентаре идут маленькими иконками;
   - `Smoke x18` отображается как 3 иконки по 6;
   - `M249 x2` отображается как 2 отдельные иконки;
   - чай отображается вместо дронов;
   - при `Забрать выбранное` в OBS появляется агрегированный визуал.

## Важная заметка

Старые уже сохранённые записи в базе, если они раньше были склеены в один большой стак предыдущей логикой, автоматически не расклеиваются задним числом.
Новые открытия после установки патча будут сохраняться корректно — отдельными инвентарными записями и с нужной раскладкой по иконкам.
