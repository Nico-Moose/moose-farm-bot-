Этот патч делает только безопаское восстановление вкладки зданий без трогания серверной логики.

Что изменено:
- восстановлен безопасный рендер вкладки "Здания", чтобы она больше не ломала общий render профиля;
- сверху во вкладке зданий добавлен блок "Текущие ресурсы" в том же стиле, что и на главной;
- здания выведены по 3 в ряд;
- порядок зданий зафиксирован: Завод / Фабрика / Шахта / Кузница / Укрепления / Глушилка / Центр;
- убрана строка "У тебя";
- убран текст "цена пачки";
- добавлены более понятные описания эффектов зданий, включая фабрику;
- live-refresh после покупки и апа зданий сохраняется за счёт уже существующих клиентских live-action обработчиков.
