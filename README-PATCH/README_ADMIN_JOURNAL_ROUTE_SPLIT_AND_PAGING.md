# ADMIN JOURNAL ROUTE SPLIT AND PAGING SAFE PATCH

Что изменено:
- добавлен отдельный лёгкий роут `routes/adminJournalRoutes.js` для админ-журнала;
- `server.js` подключает его по пути `/api/admin/journal/*`;
- админ-журнал больше не грузит всё одним большим списком: теперь по умолчанию 100 записей;
- добавлена кнопка `Показать ещё`;
- autocomplete игроков для журнала переведён на отдельный лёгкий роут `/api/admin/journal/players`;
- старые общие `/api/admin/players` и `/api/admin/events` не ломались и не удалялись.

Что НЕ трогалось:
- рынок;
- здания;
- рейды;
- обычный журнал игрока;
- CSS.

Зачем это нужно:
- легче админ-маршрут;
- меньше нагрузка на журнал за 7 дней;
- быстрее поиск игроков для журнала;
- безопаснее будущие правки админ-журнала отдельно от остальной админки.
