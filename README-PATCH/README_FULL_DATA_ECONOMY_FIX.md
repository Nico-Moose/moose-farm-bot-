# Moose Farm economy fix

Исправлено отображение дохода на главной странице.

Главная карточка раньше показывала `profile.total_income` как `📈 Доход`. `total_income` в WizeBot — это общий накопленный доход за всё время, поэтому могли появляться цифры типа 3кк.

Теперь главная показывает `📈 Доход/ч` из `farmInfo.hourly.total`: пассив уровня + урожай растений/животных + монетные здания. Запчасти, оружие и щиты не добавляются в монетный доход.

После установки: перезапусти сервер, сделай `!синкферма`, затем Ctrl+F5.
