# PATCH: журнал пользователя — 10 на страницу, 7 дней, группировка по дням

## Что сделано
- Пользовательский журнал (`История / журнал`) теперь показывает только события текущего игрока, как и раньше по `twitch_id`.
- Добавлена выборка только за 1-7 дней, по умолчанию — 7 дней.
- Добавлена пагинация по 10 событий на страницу.
- События визуально сгруппированы по дням.
- Тексты событий расшифрованы понятнее, вместо общего `Действие выполнено.` там, где это возможно.
