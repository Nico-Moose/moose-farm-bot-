# BUILDINGS LIVE REFRESH + CLEAN UI

## Что сделано

Патч дорабатывает раздел зданий.

### 1. Живое обновление зданий без F5

После покупки или апгрейда здания сайт теперь сразу применяет ответ action-endpoint и перерисовывает UI.

Затронуто:
- покупка здания
- ап здания +1
- ап здания +10

После успешного действия дополнительно запускается тихий `loadMe(true)` для добивки консистентности.

### 2. Верхняя строка ресурсов в разделе зданий

В раздел зданий добавлена такая же строка ресурсов, как на главной:
- голда
- ферма
- бонусные
- запчасти

### 3. Убрана колонка `У тебя` из карточек зданий

В карточках зданий больше не показывается отдельный блок:
- `У тебя ...💰`
- `...🔧`

Теперь ресурсы смотрятся сверху в общей строке.

### 4. Убран текст про `цену пачки`

Текст `цена пачки` не используется.
Для +10 показывается компактно:
- сколько уровней доступно
- сколько нужно ресурсов
