Moose Farm safe-patch: донат-лут в профиле и админские действия

Что добавлено
- отдельный сайтовый модуль донат-лут системы:
  - донат-баланс
  - открытие донат-кейсов 100/200/300/500
  - промокод
  - инвентарь с выдачей предметов
  - история последних выдач
- админские действия на сайте:
  - начислить донат-баланс игроку
  - откатить последнее забрание игрока
- компактный блок в профиле под статусом игрока:
  - кнопка донат-баланса
  - кнопка инвентаря
  - большое всплывающее окно с кейсами, промокодом и списком предметов

Замечания
- основная farm-логика не менялась
- farm_v2 / bridge / рынок / здания / рейды не затронуты по логике
- донат-лут хранится отдельно от обычной фермы
- открытие кейсов тратит только отдельный donate balance, не трогает farm/twitch/upgrade balances фермы

По визуальному эффекту через WizeBot
- в код добавлен безопасный hook через Twitch chat bot
- чтобы сайт отправлял команду на WizeBot visual effect, можно задать env:
  - WIZEBOT_LOOT_OPEN_COMMAND
  - WIZEBOT_LOOT_TAKE_COMMAND
- если env не заданы, сайт работает без ошибки, просто без автотриггера визуала
- аргументы отправляются в чат как:
  - <COMMAND> <login> <urlencoded_label> [<urlencoded_case_name>]
- это сделано как safe-заготовка, чтобы привязать к уже существующей WizeBot-команде/overlay у тебя без поломки сайта

Что проверить руками
1. Войти под обычным игроком, открыть /farm
2. В профиле должны появиться кнопки Донат и Инвентарь
3. Открыть модалку, активировать промокод nico_moose
4. Открыть кейс 100/200/300/500 при достаточном donate balance
5. Забрать предмет целиком по кнопке Забрать
6. Забрать частично через кнопку Часть / по названию
7. В админке:
   - выбрать игрока
   - начислить донат
   - выполнить откат последнего забрания
