# Admin: вкладка «Все фермеры» в формате журнала + жёсткое удаление фермера

Что сделано:
- добавлена отдельная вкладка `Все фермеры`
- верхний блок выбора игрока и редактор скрываются при открытии вкладки `Все фермеры`, как у журнала
- внутри вкладки добавлен собственный поиск по нику, сортировка и обновление списка
- клик по карточке фермера открывает игрока в редакторе и переключает на `Расширенное`
- кнопка `Удалить фермера` теперь удаляет запись из `farm_profiles`, поэтому игрок исчезает из списка фермеров и должен снова покупать ферму

Изменённые файлы:
- `public/farm.html`
- `public/js/10k-admin-unified-editor.js`
- `public/css/07-admin-unified-editor.css`
- `routes/adminRoutes.js`
