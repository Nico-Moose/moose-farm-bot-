# Hotfix: рынок, здания, админка

Исправлено:

1. Рынок
- поле количества теперь принимает сокращения: `1к`, `10к`, `100к`, `1кк`, `100кк`;
- быстрые кнопки заменены на `1к`, `10к`, `100к`, `1кк`, `макс купить`, `макс продать`;
- калькулятор пересчитывает стоимость с учетом сокращений;
- операции рынка отправляют на сервер уже числовое количество.

2. Здания
- убран лишний большой блок-заголовок во вкладке зданий;
- оставлены только карточки зданий;
- карточки сделаны читабельнее: меньше узких колонок, лучше переносы, понятнее требования, цена, баланс, хватит на N уровней.

3. Админка
- исправлена ошибка `Maximum call stack size exceeded`;
- `renderAdminPlayer` больше не вызывает сам себя через wrapper;
- backup-блок добавлен без рекурсивного вызова.
