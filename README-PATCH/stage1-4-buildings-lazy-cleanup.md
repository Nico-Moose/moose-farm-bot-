# Stage 1.4 — buildings lazy cleanup (safe patch)

Что изменено:
- `public/js/01-core-main.js`
- `public/js/10b-case-tabs-loader.js`
- `public/js/10c-buildings-combat-ui.js`

Что сделано:
- вкладка `Здания` больше не перерисовывается скрыто на каждом `loadMe()`;
- добавлен безопасный helper `refreshBuildingsIfVisible(force)`;
- при открытии вкладки `buildings` рендер выполняется сразу из актуального `state`;
- добавлены `dataset.loaded`-метки для блока ресурсов и контейнера зданий, чтобы не гонять лишний перерендер, пока вкладка скрыта.

Что НЕ менялось:
- серверные маршруты;
- покупка/ап зданий;
- сортировка и UI карточек;
- рынок;
- боевая механика.

Зачем:
- убрать legacy-поведение, где hidden-вкладка зданий постоянно перерисовывалась даже когда игрок её не открывал;
- снизить лишнюю работу фронта при обычном `loadMe()` и после действий вне вкладки зданий.

Риск:
- низкий, потому что изменена только стратегия вызова рендера, а не сама механика зданий.
