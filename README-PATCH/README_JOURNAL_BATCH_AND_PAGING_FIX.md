# PATCH: журнал 10 событий, страницы, расшифровка и один лог на одно админ-действие

## Что исправлено
- Пользовательский журнал показывает 10 событий на странице без внутренней прокрутки блока.
- В пагинации явно отображается `Страница X из Y`.
- Журнал хранится 7 дней вместо старого ограничения `10 событий каждого типа`.
- `Сохранить все поля` в админке теперь отправляет один batch-запрос и создаёт одно событие `admin_set_fields_batch`.
- `Сохранить все здания` теперь отправляет один batch-запрос и создаёт одно событие `admin_set_buildings_batch`.
- В пользовательском журнале и админ-журнале добавлена расшифровка:
  - какое поле изменено;
  - какое значение установлено;
  - какое здание и на какой уровень поставлено.
