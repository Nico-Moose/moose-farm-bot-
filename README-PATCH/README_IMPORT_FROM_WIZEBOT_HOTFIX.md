# Import from WizeBot Hotfix

Дата: 2026-05-03

Что исправлено:
- Кнопка `Импортировать игрока из WizeBot` больше не требует, чтобы игрок уже заходил на сайт.
- Если профиля на сайте нет, админка создаёт локальный профиль с техническим `twitch_id = wizebot:<login>`.
- После создания профиль сразу наполняется данными из WizeBot.
- Сообщение админки теперь показывает нормальный текст успешного импорта.
- Ошибки админ-действий показывают больше деталей.

Важно:
- Это не меняет реальную Twitch-авторизацию игрока.
- Когда игрок потом зайдёт через Twitch, можно будет отдельно делать перенос/слияние, если twitch_id отличается.
