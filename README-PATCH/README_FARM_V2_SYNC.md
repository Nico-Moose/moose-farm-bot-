# FARM V2 sync patch

Что добавлено:
- `GET /bridge/farm-v2-state?login=...&secret=...` — отдаёт актуальный `farm_v2` из профиля сайта.
- Команда WizeBot `!v2sync [login]` — забирает `farm_v2` с сайта и пишет в `farm_v2_<login>`.
- Команды `!ферма2` и `!мани2` читают новый формат `farm_v2_<login>`.

## Как использовать
1. На сайте играешь как обычно.
2. В WizeBot вводишь `!v2sync`.
3. Проверяешь `!ферма2` и `!мани2`.

Это переходный вариант, пока старая ферма ещё живёт на `get_var`.
