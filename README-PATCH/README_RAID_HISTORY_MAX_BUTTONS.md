# Raid History + Max Farm Buttons Patch

Дата: 2026-05-03

Что сделано:
- Кнопки `Улучшить ферму +1` и `Улучшить ферму +10` затемняются и становятся disabled, если ферма уже на максимальном уровне.
- Под кнопкой `Рейд` добавлена маленькая аккуратная кнопка `История рейдов`.
- Кнопка истории показывает последний рейд и открывает красивую сводку.
- В модалке можно переключать последние 10 рейдов.
- Сводка рейда показывает дату, атакующего, цель, силу, доход, щит, украденные монеты, бонусные, множитель и турельные поля, если они есть в логе.

- Сделать отдельную вкладку полной истории рейдов за 14/30 дней.
