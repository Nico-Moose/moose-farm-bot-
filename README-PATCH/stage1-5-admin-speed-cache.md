# Stage 1.5 — safe speed patch for admin lookup/events

Что изменено:
- `routes/adminRoutes.js`

Что сделано:
- для `/api/admin/players` добавлен короткий cache по `prefix`;
- поиск игроков переведён на заранее подготовленный statement;
- для `/api/admin/events` добавлен короткий cache по `(login, type, days, limit)`;
- выборка событий переведена на 4 prepared statements вместо динамической сборки SQL каждый раз;

- логика игровых маршрутов;
- рынок, здания, рейды, кейсы, farm_v2 sync.
