# stage1-18-buildings-tab-recover-render

## Что сломалось
После серии safe-cleanup вкладка `Здания` могла открываться пустой: заголовок и контейнер есть, а карточки зданий не дорисовываются.

## Причина
Фронт уже перешёл на lazy-поведение вкладок, но у `buildings` не осталось безопасного `re-render` при открытии вкладки. Если начальный `loadMe()` не успевал дорисовать финальный buildings-рендер или контейнер оставался пустым, вкладка сама не восстанавливалась.

## Что исправлено
- добавлен `refreshBuildingsIfVisible(force)`;
- при `loadMe()` на активной вкладке `buildings` теперь принудительно дорисовывается блок зданий;
- при `openFarmTab('buildings')` здания дорисовываются из текущего `state`, даже если начальный hidden-render не сработал.

## Что не менялось
- рынок;
- серверные маршруты;
- механика покупки/апа зданий;
- боёвка.
