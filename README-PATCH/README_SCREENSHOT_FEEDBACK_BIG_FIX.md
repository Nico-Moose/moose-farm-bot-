# Screenshot feedback big fix

Патч закрывает 5 блоков замечаний со скриншотов.

## 1. Карточки зданий

- Убрано дублирование `Статус` / `Стопор` / лишние повторяющиеся пояснения.
- Здания отсортированы логически:
  1. Завод
  2. Фабрика
  3. Кузница
  4. Шахта
  5. Укрепления
  6. Глушилка
  7. Центр
- Карточки стали компактнее и понятнее:
  - требование;
  - статус;
  - следующий уровень;
  - цена;
  - ресурсы игрока;
  - сколько уровней реально хватит;
  - польза следующего уровня.
- Адаптив исправлен: карточки не должны ломаться и съезжать на средних ширинах.

## 2. Рейды / турель

- Модалка рейда стала короче.
- Главное вынесено наверх:
  - итог монет;
  - бонусные;
  - сила атаки.
- Подробности оставлены в раскрываемом блоке.

## 3. 409 Conflict

- Добавлены frontend-locks на игровые POST-действия.
- Кнопки на время запроса блокируются и показывают `Выполняется...`.
- `409 action_in_progress` теперь показывается пользователю нормальным текстом и вызывает обновление профиля.
- Это защищает от спама кликов по защите, турели, рейд-силе и зданиям.

## 4. UI-адаптив

- Добавлены новые классы для карточек зданий и боевого блока.
- Исправлены узкие карточки, где цена/кнопки разъезжались.
- Кнопки стали ровнее и не должны залезать на соседние элементы.

## 5. Приветствие

- Заголовок `Моя ферма` заменяется на приветствие по времени суток:
  - Доброе утро
  - Добрый день
  - Добрый вечер
  - Доброй ночи
- Имя берётся из профиля.
