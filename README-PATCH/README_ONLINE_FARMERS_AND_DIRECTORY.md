# PATCH: онлайн-фермеры слева + список всех фермеров + popup профиля

## Что добавлено
- Слева на сайте добавлен блок `🟢 Онлайн фермеры`.
- Появилась кнопка `👥 Все фермеры`, которая открывает popup со всем списком игроков.
- В popup списка есть поиск по нику, сортировка по алфавиту и метка онлайн/оффлайн.
- По клику на любого игрока открывается отдельный popup-профиль.
- В popup-профиле показываются:
  - аватар
  - уровень
  - голда / ферма / бонусные / запчасти
  - лицензия / защита / рейд-сила
  - здания

## Как определяется онлайн
- Пока пользователь сидит на сайте, фронт шлёт heartbeat каждые 30 секунд.
- Онлайн считается тот игрок, у которого `last_seen_at` свежее 3 минут.
