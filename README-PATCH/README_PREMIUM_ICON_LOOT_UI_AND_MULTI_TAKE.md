# Premium loot inventory UI + icon inventory + multi-take

Патч сделан поверх архива `moose-farm-bot--main (30).zip`.

## Что изменено

### UI / визуал
- Полностью обновлён внешний вид донат-инвентаря в премиум тёмно-зелёном стиле.
- В профиле добавлены красивые карточки:
  - донат-баланс
  - количество предметов
  - кнопка поддержки `Donatex`
- В модальном окне добавлена кнопка поддержки со ссылкой:
  - `https://donatex.gg/`
- Текст `Сайт повторяет логику !лут: доступны кейсы ...` убран.
- В блоке кейсов оставлены только кнопки `100 / 200 / 300 / 500 ₽`.

### Инвентарь
- Инвентарь теперь рендерится не связками-карточками, а иконками предметов.
- Каждый предмет в связке разбивается на отдельную иконку-плитку.
- Игрок может выбрать один или несколько предметов.
- Выбранные предметы подсвечиваются.
- Добавлены:
  - панель выбора
  - суммарные выбранные предметы
  - кнопка `Забрать выбранное`
  - кнопка `Сбросить`

### Логика забирания
- Добавлен новый API endpoint:
  - `POST /api/loot/take-selection`
- Он принимает массив выбранных предметов и забирает их одним действием.
- После забирания формируется агрегированный `prizeLabel`, например:
  - `M249 x2`
  - `M249 x2 + Jagger x1`
- OBS overlay продолжает работать через существующий event-bus `loot_take`.
- WizeBot overlay/chat trigger не используется.

## Изменённые файлы
- `services/lootService.js`
- `routes/apiRoutes.js`
- `public/js/10n-loot-profile.js`
- `public/css/10zzzz-loot-profile.css`

## Что сделать после замены
1. Заменить файлы из патча.
2. Перезапустить сайт / Node процесс.
3. Открыть профиль игрока и проверить модалку донат-инвентаря.
4. Проверить:
   - открытие кейсов 100 / 200 / 300 / 500 ₽
   - выбор нескольких иконок
   - кнопку `Забрать выбранное`
   - появление визуала в OBS

## Примечание
- Старый одиночный API `POST /api/loot/take` сохранён, но новый UI использует `POST /api/loot/take-selection`.
- Логика farm / market / raids / bridge не менялась.
