# PATCH: плавающее перетаскиваемое окно Twitch-чата

## Что сделано
- Для блока Twitch-чата на главной странице добавлены кнопки:
  - `✋ Переместить` — включает свободное плавающее окно
  - `📌 Закрепить` — возвращает чат в обычный правый блок
- В свободном режиме окно чата можно перетаскивать мышкой за шапку.
- Позиция окна сохраняется в `localStorage` и восстанавливается после перезагрузки страницы.
- На мобильных устройствах плавающий режим автоматически отключается, чтобы не ломать адаптив.
