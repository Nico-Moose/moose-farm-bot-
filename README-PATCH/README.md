# Патч журнала истории

Что изменено:

- Для рейдов в журнале теперь показывается понятная строка с итогом:
  - успешный рейд: `+монеты` и `+бонусные монеты`;
  - рейд, отбитый турелью: отрицательное списание, если оно есть.
- Для оффсбора в журнале теперь показывается, сколько получено:
  - ферма/монеты оффсбора;
  - запчасти;
  - период оффлайна, если он есть в событии.
- Убрано общее `Действие выполнено.` для рейдов и оффсбора.
- Журнал теперь хранит только последние 10 записей каждого типа события для каждого игрока, чтобы база не разрасталась.

Изменённые файлы:

- `public/app.js`
- `services/userService.js`

Ничего по рынку, зданиям, рейдам как механике, балансам и sync не менялось.
