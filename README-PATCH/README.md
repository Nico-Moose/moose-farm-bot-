# Admin journal safe patch

Точечный патч только для журнала в админ-панели.

Что изменено:
- админ-журнал грузит события за последние 7 дней;
- можно смотреть журнал всех игроков или указать ник конкретного игрока;
- при открытии вкладки "Журнал" скрывается карточка данных игрока и поиск игрока сверху, остаётся только журнал;
- в фильтр журнала добавлены игровые события: рейды, оффсбор, рынок, кейсы, GAMUS, апы;
- формат строк журнала стал человеческий: суммы, цель, куплено/продано, получено;
- остальные вкладки админ-панели и рынок не трогались.

Файлы для замены:
- public/app.js
- public/farm.html
- routes/adminRoutes.js
