# PERFORMANCE FULL BOOST SAFE PATCH

Дата: 2026-05-04

Точечный safe-patch для ускорения сайта без изменения игровой логики.

## Что добавлено

### 1. Compression
- В `server.js` добавено gzip-сжатие через `compression`.
- Если пакет ещё не установлен, сервер не падает: compression включится после `npm install`.
- В `package.json` добавлена зависимость `compression`.

### 2. Быстрый серверный кэш API
- Новый файл: `services/apiCacheService.js`.
- Кэшируется:
  - `/api/me` примерно на 1.2 сек с учётом `updated_at` профиля;
  - `/api/farm/info` примерно на 1.5 сек;
  - `/api/farm/top` примерно на 5 сек.
- После любых POST-действий фермы кэш игрока и топов сбрасывается.
- Игровые операции и балансы не менялись.

### 3. Индексы SQLite
- Добавлены индексы для ускорения:
  - поиска игрока по login;
  - журнала по игроку/типу/дате;
  - общего журнала по дате;
  - журнала по типу/дате.

### 4. Frontend debounce / coalescing
- Новый файл: `public/js/11-performance-boost.js`.
- Повторные одновременные `loadMe`, `loadHistory`, `loadTops`, `loadAdminEvents` склеиваются в один запрос.
- Поиск игроков в админке получает debounce 180 мс.

## Что НЕ трогалось
- Рынок и его логика.
- Здания.
- Рейды.
- История/журнал логика.
- CSS.
- WizeBot currency как источник правды.
- Async sync не меняет `updated_at`.

## Важно
После установки патча желательно выполнить `npm install`, чтобы подтянулся пакет `compression`.
