# ADMIN PANEL MODULE SPLIT SAFE PATCH

Патч выносит JS-логику админ-панели из `public/app.js` в отдельный файл:

- `public/js/admin-panel.js`

Что изменено:

- админ-панель, действия админки, поиск игроков, админ-журнал и его autocomplete вынесены в отдельный JS-файл;
- `public/farm.html` подключает новый файл после `/app.js`;
- `public/app.js` оставлен с общей логикой сайта/фермы/рынка/истории;
- CSS не трогался;
- HTML админ-панели не переписывался;
- API и backend не трогались.

Важно:

- Поведение должно остаться 1 в 1 как в текущем архиве.
- Это только безопасное разделение кода для дальнейшего удобного редактирования админки.
