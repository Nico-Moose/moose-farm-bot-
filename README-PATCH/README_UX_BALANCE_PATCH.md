# UX Balance Patch

Что изменено:

- Добавлена липкая панель текущих ресурсов под профилем:
  - общий доступный баланс для действий на сайте;
  - баланс фермы;
  - ап-баланс;
  - запчасти;
  - стоимость следующего апа фермы и нехватка.

- В карточках зданий теперь сразу видно:
  - цена покупки или следующего улучшения;
  - сколько монет и запчастей есть сейчас;
  - сколько не хватает.

- Сообщение результата действия стало липким снизу экрана и поддерживает несколько строк.
  Теперь при ошибке апа/покупки видно причину, текущий баланс, цену и нехватку без прокрутки.

- Ошибки покупки зданий и лицензий теперь возвращают свежий профиль в API, чтобы фронт мог показать точные цифры.
