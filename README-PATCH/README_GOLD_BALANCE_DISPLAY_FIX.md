# Gold balance display fix

Исправляет отображение обычных WizeBot-монет после `!синкферма`.

Теперь на главной есть отдельные строки:

- `💰 Голда` = `twitch_balance` из `currency get` / `!мани`
- `🌾 Ферма` = только `farm_balance`
- `💎 Бонусные` = только `upgrade_balance`
- `💳 Доступно для трат` = сумма `💰 + 🌾 + 💎`

После установки обязательно:

1. перезапустить Node.js сервер;
2. в Twitch выполнить `!синкферма`;
3. в браузере нажать `Ctrl+F5`, чтобы сбросить старый `public/app.js` из кэша.

Если в чате бот пишет `сайт-ферма обновлена: ... 💰910825`, значит backend получил голду. Если в браузере всё ещё 0 — это почти всегда кэш старого JS.
