# Buildings separate resources + live upgrade fix

Что исправлено:
- во вкладке `Здания` блок `Текущие ресурсы` вынесен в отдельную панель между вкладками и карточками зданий;
- в этой панели оставлены только: `Голда`, `Ферма`, `Бонусные`, `Запчасти`;
- строка `Следующий ап...` убрана из вкладки `Здания`;
- карточки зданий перестроены в сетку `3 в ряд`;
- порядок: `Завод / Фабрика / Шахта / Кузница / Укрепления / Глушилка / Центр`;
- убраны лишние блоки `У тебя` и `цена пачки`;
- кнопки покупки и апа зданий переведены на отдельный live-refresh без F5;
- после покупки/апа делается немедленный `loadMe(true)` и короткий повторный refresh.
