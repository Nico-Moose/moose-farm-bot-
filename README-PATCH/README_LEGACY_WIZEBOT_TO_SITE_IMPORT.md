# Legacy WizeBot → site/farm_v2 import

Дата: 2026-05-03

Что делает патч:
- Кнопка импорта теперь считается именно как перенос старой `!ферма` из WizeBot в сайт и новую `farm_v2`.
- Если в WizeBot нет legacy vars (`farm_`, `farm_virtual_balance_`, `farm_upgrade_balance_` и т.п.), импорт не создаёт пустую ферму и не затирает сайт нулями, а возвращает понятную ошибку.
- При переносе данные старой фермы мержатся с текущим `profile.farm`, поэтому не теряются site-only поля (рейды, кейсы, shield, cooldown и т.д.).
- После импорта профиль сразу пушится обратно в WizeBot как `farm_v2_*`, чтобы `!ферма2` и новые команды видели те же данные.

Важно:
- `!v2sync` в WizeBot — это обратная команда: она тянет farm_v2 С САЙТА в WizeBot.
- Для миграции старой `!ферма` → `!ферма2` надо использовать кнопку админки `Импортировать старую !ферму в farm_v2`.

Ожидаемый результат:
- у игрока старая `!ферма` и новая `!ферма2` после импорта показывают одну и ту же основу: уровень, farm_balance, bonus, parts, license, protection, raid_power и постройки.
