# Web-master auto push + harvest sync

Что добавлено:
- автоматический push сайт -> WizeBot после всех игровых POST-действий сайта;
- bridge endpoint `/bridge/wizebot-pull-sync` для подтягивания урожая и пассива из WizeBot на сайт;
- обновлённая команда `wizebot_commands/!урожай.txt`, которая после начисления вызывает pull-sync на сайт для каждого игрока;
- в админку добавлены кнопки: импорт из WizeBot, пуш в WizeBot, подтянуть урожай из WizeBot;
- кнопка сбора урожая на сайте переведена в режим информации: урожай управляется WizeBot-командой `!урожай`.

Что нужно сделать после выкладки:
1. Обновить `.env` и оставить `HARVEST_MANAGED_BY_WIZEBOT=true`.
2. Перезапустить сайт.
3. В WizeBot заменить код команды `!урожай` содержимым файла `wizebot_commands/!урожай.txt`.
4. Убедиться, что `WIZEBOT_BRIDGE_SECRET` совпадает в сайте и в командах WizeBot.
5. Проверить: после `!урожай` на стриме у игрока обновляются сайт и WizeBot; после любого действия на сайте `!ферма` и `!мани` показывают актуальные значения.
